# 更新内容

## GKD-X / MIUIX

- 界面全面适配 MIUIX（顶栏、底栏、设置分组、对话框、图标等）
- 应用品牌调整为 GKD-X（`li.songe.gkdx`）
- 首页按服务 / 数据概览 / 快捷入口分组
- 触发记录页适配 MIUIX 样式与字体色
- 订阅页取消下拉刷新，顶栏增加刷新按钮
- 触发提示支持实时通知，并可自定义存在时间
- Debug 构建不再给应用名追加 `-debug` 字样

## 更新方式

- GKD-X - 设置 - 关于 - 检测更新
- 或前往 [GitHub Releases](https://github.com/hanchuan8/gkd-miuix/releases) 下载
